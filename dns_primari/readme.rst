-----------------------
DNS Autoritatiu Primari
-----------------------



Configuració
------------

Totes les zones a `conf`_.

Esciure tots els serials com a 2017100101 perquè siguin canviats dinàmicament.


Execució
--------

`docker-compose up`


Instal·lar eines addicionals per a debugar:

* `apk --update add bind-tools`



Producció
---------

docker swarm init
